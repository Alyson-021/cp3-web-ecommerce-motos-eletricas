# EletroRide

Projeto de ecommerce de motos elétricas desenvolvido para a disciplina de Web Development da FIAP.

## 🚀 Tecnologias
- HTML
- CSS
- JavaScript
- DOM
- Reduce
- Git/GitHub

## ⚡ Funcionalidades

- Cards de produtos com DOM
- Carrinho de compras
- Cálculo total com Reduce
- Desconto de 10%
- Aba de Catálogo para cativar cada perfil de cliente (persona)
- Responsividade
- Navegação entre páginas

## 🧠 Conceitos aplicados

- Manipulação do DOM
- Arrays e Objetos
- forEach()
- reduce()
- Eventos
- Responsividade
- Estrutura semântica HTML5
- Organização modular
- Git Flow básico
